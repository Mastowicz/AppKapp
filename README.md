Prosta aplikacja umożliwiająca liczenie kroków, pokazująca aktualną pozycję geograficzną oraz zapisywanie tych danych. W zakres pracy wchodzi liczenie kroków, pokazywanie aktualnej pozycji geograficznej oraz zapisywanie tych danych. Aplikacja pokazuje również aktualną dokładność GPS. Pokazuje również czas startu liczenia oraz jego zakończnia. 

Instalacja (działa tylko na androidzie lub jego emulatorze):
1. Uruchom plik project1.apk
2. Zatwierdź instalację
3. Otwórz zainstalowaną aplikacje
